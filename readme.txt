== Description ==

Displays Sowprog events. Events list, location details, event details.

You'll need a <a href="https://pro.sowprog.com/signup?role=agenda" >Sowprog Agenda account</a> to run the import. This account is free and open to anyone. 

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress

You'll find a SOWPROG menu item under Tools menu.
Here you will be able to configure your Sowprog account and on which page to display the events

<a href="mailto:contact@sowprog.com">Contact us</a> or <a href="https://pro.sowprog.com/signup?role=agenda" >create an Agenda account</a> on Sowprog.
